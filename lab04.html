GDP changes across countries within the continents.

Table

part_2_2 <- gapminder %>% 
  select(continent,country,year,gdpPercap) %>% 
  group_by(country,continent) %>% 
  filter(year == 1952 | year == 2007) %>% 
  pivot_wider(names_from = year,names_prefix = "gdp_",values_from = gdpPercap) %>%
  mutate(gdp_change = gdp_2007 - gdp_1952)

Description/Writeup

Something that this figure doesn’t tell you that you would be able to get from the table is what countries experienced the change. For example we see that one Asian country has had their GDP drop significantly from the table we would know that this country is Kuwait however the boxplot doesn’t tell us this information. What this figure adds is that we are better able to see the trends. We can see that on European on average experienced higher GDP growth whereas the GDP of African countries has on average not changed at all.
